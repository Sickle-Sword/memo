reticulate パッケージ

reticulate

  • R上でpythonを動かせるパッケージ
  • VScodeとかpython用にエディタとかを入れてみたものの，結局プロジェクト管理とかが面倒になってしまったので，全部Rstudio上でできるようにしたかった
library(reticulate)

使い方

1. python環境の指定

  • 狙ったpythonのバージョンを指定する
  • ここで結構ハマった。なんでも一度何らかの関数でpythonを動かしてしまうと， 一度Restart sessionしない限りpythonの環境を変更できないらしい。
  • むしろlibrary(reticulate)よりも前にこれを実行しておいた方がいいかもしれない
reticulate::use_python('/Userse/うんたら/かんたら/Python/Python38', required = TRUE)

Sys.which('python')とかがpythonの場所を探すのに役に立った記憶

2. 対話モード

  • モチベーション的に，RstudioというIDEの下でpythonを使いたいので早速対話モードを試してみる
  • 対話モードにするにはrepl_pythonと打つ

これを実行すると，コンソールが次のように変わる

repl_python()
Python 3.8.3 (C:/Users/うんたら/かんたら/Python/Python38/python.exe)
Reticulate 1.16 REPL -- A Python interpreter in R.
>>>

最後の>>>が，コンソールがpythonコマンドを受け付けている目印になる。 通常のRに戻るときはquitと打つ。（それかスクリプトからRのコードを走らせる）

>>> quit
>
  • （Rmarkdownでrepl_pythonをチャンク上で実行するとなぜかフリーズするので注意）

3. 色々やってみる

  • まずは基本のやつ
print("hallo world")
hallo world
  • Rのコンソールの状態から，pythonコードを実行するとこんな感じになる。

データの加工

  • pythonでのデータの加工に全く慣れていないのでアレだが，ひとまずここは列と行を絞る
  • 列はSpecies以外，行は75行をランダムサンプリングする
  • Rのパイプ（ %>% ）のような書き方もできる（メソッドチェーン）
  • pythonでは括弧内でのみ自由に改行できるので，メソッドチェーンを使うところだけ括弧でくくる
iris_sample = (
   iris[['Sepal.Length', 'Sepal.Width', 'Petal.Length', 'Petal.Width']]
   .sample(n = 75)
   )
